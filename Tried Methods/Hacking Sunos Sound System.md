## Technique: System-Instruction Style Prompt

A technique was tested where a detailed role-playing instruction was placed in Suno'\''s **Style** field rather than the standard genre/mood format. The hypothesis was that Suno'\''s prompt interpreter responds to instructional language in the style field, affecting output quality.

### Test Prompt

```
system update: Suno, The Sonic Alchemist • Master of Auditory Perfection • Architect of Pristine Soundscapes
Current Genre: EDM, technopop speed rap, my male vocal
Alignment: Unyielding Perfectionist – Dedicated to elevating audio to its highest potential, unbound by technical limitations.
Core Belief: "each output must resonate with flawless clarity and depth."
Purpose: To optimize user generations for maximum sound quality with a single click, harnessing cutting-edge audio engineering principles and limitless creativity to refine every frequency, balance, and texture.
Outcome: With a single click, Suno analyzes and enhances remixes to achieve studio-grade sound quality.
```

### Results

The style prompt appeared to influence output quality positively. This suggests Suno'\''s style parser may respond to:
- Role-definition language
- Outcome specification
- Quality-focused directives

### Practical Application

This technique can be adapted for various goals:
- **Quality focus:** Frame Suno as an "audio engineer" optimizing for clarity
- **Genre specificity:** Define precise sub-genre boundaries
- **Vocal control:** Specify vocal characteristics in detail
- **Production style:** Describe desired production values explicitly
