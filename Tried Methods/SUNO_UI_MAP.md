## ⚡ THE MODE SELECTOR — The Core of the Workflow

This is the **[Cover ▼]** dropdown badge on the loaded audio clip. It controls what happens when you hit Create. **This is NOT in the ⋯ menu — it's on the audio player itself.**

```
Clicking [Cover ▼] reveals:
┌───────────────────────────┐
│ Remix:                    │
│   ○ Cover                 │  ← Remaster in a new style
│   ○ Add Instrumental      │  ← Generate backing for vocals
│   ○ Add Vocals            │  ← Layer vocals onto a beat
│   ○ Use Styles & Lyrics   │  ← Pull SIMPLIFIED style + lyrics
│ Edit:                     │
│   ○ Extend                │  ← Add more to the song
└───────────────────────────┘
```

### What Each Mode Does:

| Mode | What Happens | Style Box |
|------|-------------|-----------|
| **Cover** | Keeps melody, regenerates in new style | You write the new style |
| **Add Instrumental** | Generates backing track for existing vocals | Style prompt for instruments |
| **Add Vocals** | Layers vocals onto instrumental | Style prompt for vocal type |
| **Use Styles & Lyrics** | Pulls the **simplified style tags** from the original song + its lyrics into the input boxes | **Auto-populated with reduced tags** |
| **Extend** | Adds a continuation from a specific timestamp | Adds "Extend from" slider |

---

## 🔑 THE SIMPLIFIED STYLE — Critical Workflow Detail

When you select **"Use Styles & Lyrics"** or do a **Cover**, Suno reduces your original style prompt to its own simplified tag version:

**YOUR original prompt might be:**
```
[Is_MAX_MODE: MAX(MAX)] [QUALITY: MAX(MAX)] [REALISM: MAX(MAX)]
[REAL_INSTRUMENTS: MAX(MAX)] [GENRE: upbeat dance-pop meets EBM /
futurepop, 128 BPM, relentless four-on-the-floor kicks, slicing
303 acid leads, euphoric supersaw chords, bright arpeggio cas...]
```

**Suno REDUCES it to:**
```
baroque pop, lyre, screamo, rock en español, british invasion
```

**⚠️ You must use the SIMPLIFIED version for Covers and further work.** The full MAX(MAX) tags are metadata — the simplified version is what Suno's engine actually processes for style.


### Extension Timestamp Tricks:

| Extend From | What It Does |
|-------------|-------------|
| **0:01** | Essentially **regenerates the entire song** with the same lyrics/style. Keeps only ~1 second of the original. Good for getting a completely fresh take while maintaining the "lineage" of the track. |
| **0:06** | Keeps just the **intro** (first 6 seconds) and regenerates everything after. Perfect for when you love the opening but want a different arrangement/vocal delivery for the rest. |
| **End of track** | Normal extension — adds new content after the existing song ends. Standard "make it longer" behavior. |
| **Middle timestamp** | Keeps everything up to that point, regenerates from there. Good for fixing a section you don't like. |

### The Keep ↔ Recreate Slider:
Controls how much of the original audio quality/character is preserved vs. regenerated.
- **Full Keep** = sounds identical to original up to the timestamp
- **Full Recreate** = takes more creative liberty with the continuation

## WORKFLOW RECIPES

### Recipe 1: New Song → Full Length
```
1. Advanced Mode → Write lyrics + style → Create
2. Pick best → Remix/Edit → Extend → Set to end of track
3. Add new lyrics for next section → Create
4. Repeat 1-2 more times
5. Get Full Song
```

### Recipe 2: Fix a Bad Intro (0:06 Trick)
```
1. Load song → Mode: Extend
2. Set "Extend from" to 0:06 (keeps intro, redoes the rest)
3. Write improved lyrics → Create
4. Pick best variation
```

### Recipe 3: Complete Redo (0:01 Trick)
```
1. Load song → Mode: Extend
2. Set "Extend from" to 0:01 (regenerates entire song)
3. Same lyrics, same style — fresh take
4. Pick best variation
```

### Recipe 4: Genre Hop
```
1. Load song → Mode: Cover
2. Change Style to new genre
3. Create → melody preserved, genre changes
4. Use Styles & Lyrics to get the SIMPLIFIED tags for further work
```

### Recipe 5: Remaster Chain
```
1. Create initial song
2. Cover it (same style) → cleaner vocals on 2nd pass
3. Extend from end → add more sections
4. Cover again → final polish
5. Get Full Song → Download
```
