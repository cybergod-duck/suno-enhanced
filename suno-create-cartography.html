6
Extend
Highest Leverage
⏱
Extend @ 0:01 — Blank Style
The single highest-leverage action. Discards all context after second 1. Forces acoustic DNA inheritance: key, tempo, production texture. Overrides default context window.
Step 2 of 6-step path Style = blank Lyrics = identical
🔒
Extend @ 0:06 — Vocal Lock (Optional)
Step 2.5. Only when Step 2 has strong instrumentals but weak vocals. Locks vocal performance. Lyrics only, style blank.
Optional Step 2.5 Style = blank Lyrics = new or identical
⏭
Extend from End: Nothing
Step 4. Both fields BLANK. v5.5 metadata re-index. Credits refunded. Must run on fully compiled Get Whole Song output — not on an uncompiled Cover.
Credits refunded Lyrics = blank Style = blank
⚠ Node Dependency Warning Running end-extend on an uncompiled Cover registers an incomplete dependency tree. Stems, Voices, and v5.5 feature flags inherit from the wrong node. Always run on Get Whole Song compiled output.

Cover & Remaster
Polish Layer
🔇
Cover: No Style (Step 3)
The step most people skip. Waveform-only remaster. Strips spectral crowding, phase seams, mix compression that survive export otherwise. No prompt engineering recovers this post-download.
Step 3 of 6-step path Style = blank Lyrics = identical
🎙

Re-paste identical tagged lyric sheet from Step 1. Vocal-only style tokens. Sets dynamic range ceiling. Zero instrument keywords.

Remaster (Step 6)
True export ceiling. Strips synthetic HF noise and chain compression. ~5 credits, NOT refunded. Mandatory final step.
